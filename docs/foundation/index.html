# Color scheme

We have already carefully selected the optimal color scheme for Oart UI, but your business may need another color palette. Here’s how you can change the colors using Sass.

Go to the scss/oart/_variables.scss file and search for the following variables:

$primary: #174f84;
$secondary: #10ab7c;
$tertiary: #FF7F66;

Change the hex color for each variable to the color palette that you desire.
